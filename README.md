# GDG_WEB3 — Training Progress & Practice Code

**Overview:**  
This repository contains practice code and exercises from the GDG Web3 training sessions. It documents learning progress, example smart contracts, and related resources used while experimenting with Web3 development.

---

## Status

- **Progress:** Ongoing — actively practicing Solidity and smart contract development through weekly exercises.  
- **Focus Areas:** Solidity basics, smart contract deployment, testing, and local development workflows.

---

## Learning Goals

- **Understand:** Core Solidity syntax and contract structure.  
- **Practice:** Deployment flows, gas considerations, writing tests, and debugging.  
- **Experiment:** Build small dApps and connect contracts to simple frontends.
